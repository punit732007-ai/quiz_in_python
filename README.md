# Bollywood Trivia Game 🎬

A simple and fun text-based quiz game built in Python to test your knowledge of Bollywood movies!

## 🚀 Features
- 5 exciting Bollywood trivia questions.
- Built-in scoring system.
- Playable directly from the terminal or command prompt.
- Fun and interactive messages based on your final score.

## 🛠️ Prerequisites
- You need to have **Python 3.x** installed on your system.
- No external libraries are required. It uses Python's built-in `time` module.

## 🎮 How to Play
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/punit732007-ai/quiz_in_python.git
   ```
2. Navigate to the project directory:
   ```bash
   cd quiz_in_python
   ```
3. Run the Python script:
   ```bash
   python newproject.py
   ```
4. Read the questions carefully and type your answers.
5. See if you get the "Super Hit" or "Flop" rating at the end!
